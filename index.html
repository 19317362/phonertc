Open Source

PhoneRTC is completely free and open source under the Apache License.

Cross Platform

PhoneRTC has full Android, iOS and Browser support.

No third parties

You don't need to rely on any third parties in order to use PhoneRTC. Install your own servers!

Simple Javascript API

PhoneRTC provides an easy-to-use, consistent JavaScript API for all platforms.

Demo App

PhoneRTCDemo is a simple demo app for Android and iOS to demonstrate the PhoneRTC library. It supports voice call, video call and group video call. The UI is based on Ionic Framework. The full source code for the app and the signaling server is available at GitHub.
